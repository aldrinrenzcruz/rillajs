Writing Vanilla JS Made Easy

Rilla is a minimalistic JavaScript utility library designed to simplify DOM selection, manipulation, and event handling.

Disclaimer: This early build (v0.1) is still in development and may undergo significant changes without prior notice, potentially breaking your code. Use it for testing purposes only.
