Getting Started

Rilla is a minimalistic JavaScript utility library designed to simplify DOM selection, manipulation, and event handling.

Disclaimer: The current build (v0.1) is unstable and may contain bugs.

CDN

Use the jsDeliver CDN to access the latest minified version of Rilla.js:

Alternatively, you can also download the latest minified version:

  • Download the uncompressed development version of Rilla.js
  • Download the map file for Rilla.js

Installation

Add the rilla.min.js script tag in the <head> of your HTML file, and start using Rilla to write less and do more.
