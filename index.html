Getting Started

Rilla is a minimalistic JavaScript utility library designed to simplify DOM selection, manipulation, and event handling.

Download Rilla.js

Installation

Include the rilla.min.js file in your project: 

<script src="path/to/rilla.min.js"></script>

Usage

DOM Selection

Rilla.js makes it easy to select DOM elements using CSS selectors or IDs. 

// Select by ID or Class selector (returns single element or NodeList)
const element = $('#container');

// Select within an element
const childElement = element.$('.child');

// Get element by ID
const elementById = $id('container');

DOM Manipulation

Easily modify DOM elements with these utility methods. 

// Create a new element
const button = $create("button")
  .text("Copy")
  .addClass("px-2 py-1 bg-gray-800 text-gray-400");

// Append HTML to an element (returns the element for chaining)
element.$append('<div class="new-item">Content</div>');

// Prepend HTML
element.$prepend('<header>New Header</header>');

// Replace all content
element.$paint('<div>New Content</div>');

// Single wrapping or Chain wrapping
$("#container")
  .wrap("<section class='test-class'></section>")
  .wrap("<blockquote></blockquote>")

// Wrap multiple elements
$("p")
  .wrap("<blockquote></blockquote>")

// Unwrapping elements
$("p").unwrap()
$("#container").unwrap()

// Get or set HTML content
element.html();         // Get HTML content
element.html('<p>New content</p>');  // Set HTML content

// Get or set text content
element.text();         // Get text content
element.text('New text');  // Set text content

// Get or set input value
element.val();          // Get value
element.val('New value');  // Set value

Element References

Capture element references while maintaining chainability. 

// Capture element reference within a chain
element.$this(function(el) {
  // Access the element inside the callback
  console.log(el);
});

// Event handlers that modify the element
$create("button")
  .text("Copy")
  .$this(button => 
    button.$click(() => {
      button.text("Copied!");
      setTimeout(() => button.text("Copy"), 1500);
    })
  )
  .addClass("primary");

// Use with conditional logic
element
  .addClass("card")
  .$this(el => {
    if (condition) {
      el.addClass("special");
    }
  })
  .appendChild(contentElement);

Attributes and Classes

Work with element attributes and CSS classes seamlessly. 

// Get or set attributes
element.$attr('data-id');  // Get attribute
element.$attr('data-id', 'new-value');  // Set attribute

// Add classes
element.addClass('active highlighted');

// Remove classes
element.removeClass('highlighted');

// Toggle classes
element.toggleClass('active');

Event Handling

Manage event listeners with chainable methods. 

// Click event with callback function
element.$click(function(event) {
  console.log('Clicked!', this);  // 'this' refers to the element
});

// Prevent default behavior
element.$submit(function(event) {
  // Do something
}).prevent();

// Stop propagation
element.$click(function(event) {
  // Do something
}).stop();

// Chain prevent and stop
element.$click(function(event) {
  // Do something
}).prevent().stop();

// Remove event listener
element.$removeClick(callbackFunction);

Document and Window Events

Listen for document and window events. 

// DOM ready event
$dom(function() {
  console.log('DOM fully loaded');
});

// Window load event
$window(function() {
  console.log('Window loaded');
});

Show/Hide and Fade Effects

Control element visibility with simple functions. 

// Show element (accepts element or ID)
show('elementId');  // Default display is 'block'
show('elementId', 'flex');  // Custom display value

// Hide element
hide('elementId');

// Toggle visibility
toggle('elementId');

// Fade effects
fadeIn('elementId');
fadeOut('elementId');
fadeToggle('elementId');

Storage Utilities

Easily interact with browser storage. 

// Local storage
$local('key', 'value');  // Set value
const value = $local('key');  // Get value
$local('key', null);  // Remove value

// Session storage
$session('key', 'value');  // Set value
const sessionValue = $session('key');  // Get value
$session('key', null);  // Remove value

Draggable Elements

Make elements draggable with simple class names. 

<div class="draggable">
  <!-- Add a header element with the class "draggable-handle" -->
  <div class="draggable-handle">Drag me</div>
  <!-- Content -->
</div>

Global Usage in HTML Attributes

To use the library functions inside HTML attributes like onclick, make sure to use the global reference: 

<button onclick="toggle(window.$('#container'))">Toggle</button>
